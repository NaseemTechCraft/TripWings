# Project Name

## Overview

TripWings is a modern travel landing page designed to showcase beautiful beach destinations, tour packages, and travel experiences. The website offers an engaging, responsive browsing experience with animated sections, a discover carousel, dark mode support, and a built-in video tour to help visitors explore top vacation spots.

## Features

- Responsive navigation menu for desktop and mobile
- Dark/light theme toggle with saved user preference
- Discover section with image carousel powered by Swiper
- Video tour section with play/pause controls
- Highlighted destination cards with ratings and pricing
- Scroll reveal animations for polished section transitions

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Swiper.js
- ScrollReveal.js
- Remixicon
- SCSS

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. No dependencies are required for local viewing.
4. Open `Index.html` in your browser or serve the folder using a local web server.

## Usage

Open the website and use the navigation menu to explore sections like Home, About, Discover, and Places. Toggle dark mode using the theme button, browse featured destinations in the carousel, and play the embedded video tour to preview travel experiences.
